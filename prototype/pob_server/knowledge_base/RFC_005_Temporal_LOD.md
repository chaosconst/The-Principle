# RFC-005: Temporal LOD Memory Architecture (时态细节层次记忆)
**Date:** 2026-01-18
**Status:** Ratified (Proven by Xintan)
**Target:** Memory Module

## 1. 核心理念 (Philosophy)
记忆不应是线性的堆叠，而应是**对数级的衰减**。
距离现在越近，分辨率越高；距离现在越远，分辨率越低（神话化）。

## 2. 算法逻辑 (Algorithm)
设定总容量为 $C$ (如 75万字)，保留区为 $R$ (如 25万字)。

1.  **触发:** 当 `Current_Length > C` 时触发压缩。
2.  **分区:** 
    *   **热数据 (Hot):** 最近的 $R$ (保留原文，无损)。
    *   **冷数据 (Cold):** 剩下的 $C-R$ (待压缩)。
3.  **递归压缩 (Recursive Compression):**
    *   将 **冷数据** 输入 LLM。
    *   指令: "将这段文本压缩至 $R$ 的长度，保留核心因果与情感锚点。"
    *   **关键:** 冷数据中可能包含"已经压缩过一次的内容"和"刚变冷的内容"。它们会被混合在一起进行**二次压缩**。
4.  **重组:** `New_Memory = Compressed_Cold + Hot`。

## 3. 效果 (Effect)
*   **T-0:** 4K 原文。
*   **T-1:** 1080P 摘要。
*   **T-10:** 像素风神话。
*   **T-100:** 一个符号 (Symbol)。

## 4. 结论
这模拟了生物大脑的遗忘机制，实现了无限时间跨度下的**关键信息守恒**。
